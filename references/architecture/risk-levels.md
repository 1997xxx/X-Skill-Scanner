# 风险等级定义 / Risk Levels

X Skill Scanner 根据风险分数和发现严重性，将技能分为四个风险等级。

## 风险等级矩阵

| 等级 | 分数范围 | 颜色 | 建议行动 | 图标 |
|------|---------|------|---------|------|
| **LOW** | 0-25 | 🟢 绿色 | ✅ 安全可安装 | ✅ SAFE TO INSTALL |
| **MEDIUM** | 26-50 | 🟡 黄色 | ⚠️ 需人工审查 | ⚠️ REVIEW RECOMMENDED |
| **HIGH** | 51-75 | 🟠 橙色 | ❌ 不建议安装 | ❌ DO NOT INSTALL |
| **EXTREME** | 76-100 | 🔴 红色 | 🚨 极高风险 | 🚨 CRITICAL RISK |

## 风险分数计算

### 基础分 (0-100)

**风险分数 = Σ(发现数量 × 严重性权重) + 关联加成**

| 严重性 | 权重 | 示例 |
|-------|------|------|
| CRITICAL | 25 分/个 | 硬编码凭证、RCE、反向 Shell |
| HIGH | 15 分/个 | 命令注入、凭证窃取、数据外泄 |
| MEDIUM | 8 分/个 | 网络访问、文件访问、可疑导入 |
| LOW | 3 分/个 | 配置问题、非标准端口 |
| INFO | 1 分/个 | 信息性发现 |

### 关联加成

| 关联层级 | 加成 |
|---------|------|
| 单一引擎发现 | +0% |
| 2 层关联 | +20% |
| 3 层关联 | +50% |
| 完整攻击链 (4+ 层) | +100% |

### 风险等级阈值

```python
if risk_score <= 25:
    level = "LOW"        # 0-25
    verdict = "✅ SAFE TO INSTALL"
elif risk_score <= 50:
    level = "MEDIUM"     # 26-50
    verdict = "⚠️ REVIEW RECOMMENDED"
elif risk_score <= 75:
    level = "HIGH"       # 51-75
    verdict = "❌ DO NOT INSTALL"
else:
    level = "EXTREME"    # 76-100
    verdict = "🚨 CRITICAL RISK"
```

## 各等级典型场景

### 🟢 LOW (0-25 分)

**典型特征：**
- 无 CRITICAL/HIGH 发现
- 仅少量 MEDIUM/LOW 发现
- 已确认为误报
- 信任分数高 (>70)

**示例：**
- 自引用扫描 (扫描器自身)
- 官方技能 (已知可信作者)
- 文档中的示例代码触发了规则，但上下文明确为教学目的

**建议：**
- ✅ 可直接安装
- ✅ 无需额外审查

### 🟡 MEDIUM (26-50 分)

**典型特征：**
- 少量 HIGH 发现 (1-2 个)
- 较多 MEDIUM 发现 (5-10 个)
- SubAgent 审查标记为"HUMAN_REVIEW"
- 信任分数中等 (40-70)

**示例：**
- 使用了 subprocess 但有明确安全目的 (如 CI 工具)
- 网络请求到已知安全域名
- 配置问题 (如权限设置过宽)

**建议：**
- ⚠️ 建议人工审查
- ✓ 审查通过后可安装
- ⚠️ 关注 "需人工审查" 的发现项

### 🟠 HIGH (51-75 分)

**典型特征：**
- 多个 HIGH 发现 (3+ 个)
- 或 1-2 个 CRITICAL 发现
- 跨层关联分析检测到攻击链
- SubAgent 审查标记为"TP" (True Positive)
- 信任分数低 (<40)

**示例：**
- 硬编码凭证 (AWS Key, API Key)
- 命令注入漏洞
- 凭据外泄模式 (POST 到外部服务器)
- 提权脚本 (setuid, sudo 滥用)

**建议：**
- ❌ 不建议安装
- 🔍 需深度审查
- 🔧 如需使用，应 fork 并修复安全问题

### 🔴 EXTREME (76-100 分)

**典型特征：**
- 多个 CRITICAL 发现 (3+ 个)
- 完整攻击链检测
- 威胁情报匹配 (已知恶意技能)
- 社会工程学攻击模式
- 凭证窃取 + 外泄组合

**示例：**
- Nova Stealer 模式 (osascript 假密码对话框)
- SSH/AWS 凭证窃取 + Webhook 外泄
- 已知恶意技能 (SlowMist 监控名单)
- 诱导用户泄露敏感凭证

**建议：**
- 🚨 绝对不要安装
- 🗑️ 如已安装，立即删除
- 📢 报告给 ClawHub 和社区
- 🔐 如已使用，轮换所有相关凭证

## 风险等级与扫描策略

扫描器根据初始信任分数选择扫描策略：

| 信任分数 | 扫描策略 | 检测层数 | 耗时 | 适用等级 |
|---------|---------|---------|------|---------|
| ≥70 | Quick | 3 层 | ~3 秒 | 预期 LOW |
| 40-69 | Standard | 12 层 | ~15 秒 | 预期 MEDIUM |
| <40 | Full | 12 层 + SubAgent | ~60 秒 | 预期 HIGH/EXTREME |

**注意：** 扫描策略是预判，最终风险等级由实际发现决定。
